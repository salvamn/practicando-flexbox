HTML 5

HTML 5 es el lenguaje de etiquetas utilizado para crear y darle una estructura a nuestra web, a continuacion como observamos en la imagen de abajo podemos visualizar la estructura basica de una pagina web, no obstante la estructura mostrada no quiere decir que sea la unica, existen otras muchas estructuras, incluso hay desarrolladores de software que crean sus propias estructuras o tambien conocidas plantillas.

Estructura de una Etiqueta

En HTML 5 la estructura de una etiqueta es muy importante ya que esta tiene una sintaxis que debemos respetar siempre. Las etiquetas en HTML tiene una etiqueta de apertura y una de cierre como muestra la imagen, a demas de eso nuestras etiquetas html tienen un parametro llamado atributo que recive un valor, a veces en algunas etiquetas el atributo es opcional.


Etiquetas de Texto

<p>

La etiqueta parrafo como podemos deducir sirve para representar un parrafo. 

                            
    <p>
        Hola soy un parrafo.
    </p>

Resultado

Hola soy un parrafo

<span>

La etiqueta span es muy utilizada para aplicar estilos personalizados. 

                            
    <p> Hola soy un parrafo
        <span style="color: blue;">
            Soy personalizable
        </span>
    <p>

Resultado

Hola soy un parrafo Soy personalizable

<br>

La etiqueta br genera nada mas que un salto de linea.

                            
    <br></br>

<em>

La etiqueta em nos permite darle un enfasis al texto, en palabra mas comprensibles lo que hace es mostrar el texto o una parte del texto en cursiva. 

                            
    <em>Soy un texto en cursiva.</em>

Resultado

Soy un texto en cursiva.

<strong>

La etiqueta strong nos permite darle un estilo en negrita a una frase o palabra. 

                            
    <strong>Soy un texto en negrita</strong>

Resultado

Soy un texto en negrita

<u>

La etiqueta u nos permite subrayar un texto o palabra. 

                            
    <u>Soy un texto subrayado<u>

Resultado

Soy un texto subrayado

<mark>

La etiqueta mark nos permite resaltar un texto o frase relevante. 

                            
    <mark>Soy importante</mark>

Resultado

Soy importante

<small>

La etiqueta small nos permite cambiar el tamaño de una palabra, osea le damos un tamaño pequeño. 

                            
    <small>Soy pequeño<small>

Resultado

Soy pequeño yo no

<address>

La etiqueta address se puede usar para representar un direccion, comunmente se usa en footer. 

                            
    <address>San Pedro de la Paz<address>

Resultado

San Pedro de la Paz

Etiquetas para Imagenes

<img>

La etiqueta img nos permite poder insertar una imagen el documento, como dato esta etiqueta no tiene un etiqueta de cierra. 

                            
    <img src="imagen" alt="Soy una imagen">

Resultado

<picture>

La etiqueta picture nos sirve para mostrar imagenes en multiples resoluciones, su uso comun es al momento de hacer una web responsiva. 

                            
    <picture>
        <source srcset="imagen-42px" media="(max-width: 1024px)">
        <img src="imagen-96px">
    </picture>

Intenta cambiar el tamaño de tu navegaador y veras que un punto dado la imagen que esta abajo disminuira su tamaño.
